# OrderXpress Brand Identity
OrderXpress is a modern SaaS/consumer hybrid for QR-based restaurant management.

## Visual Language
- **Card-based Layout**: All content sits in white cards with 16px radius and soft shadows.
- **Color Strategy**: Navy (#0B3877) for data highlights, Solid Black (#111318) for primary CTAs and active navigation.
- **Status badges**:
  - Blue = New
  - Orange = Preparing/Pending
  - Green = Ready/Paid
  - Red = Reject/Delete
- **Navigation**: Bottom bar with 5 items. Active state is a black rounded-square chip.
- **Tone**: Professional, data-dense, yet appetizing for customers.
